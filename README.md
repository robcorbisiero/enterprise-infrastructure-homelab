# Enterprise Infrastructure Homelab

## Overview

This project documents the design, deployment, troubleshooting, and administration of a small enterprise-style infrastructure lab built using retired business hardware.

The lab was created to develop hands-on skills in server administration, virtualization, networking, Linux, Windows Server, hardware troubleshooting, and infrastructure documentation.

## Core Technologies

- Proxmox VE
- Ubuntu Server
- Windows Server 2016
- Hyper-V
- Active Directory
- DNS
- DHCP
- Cisco Catalyst Switching
- Cisco IOS
- PuTTY / Serial Console
- TCP/IP Networking
- SSH
- Enterprise Desktop Hardware

## Hardware

- Dell OptiPlex systems
- Lenovo ThinkCentre systems
- Cisco Catalyst switches
- Cisco Meraki MR36 access point
- SSDs, RAM, GPUs, network adapters, and power supplies

## Key Projects

### Proxmox Virtualization Host

Deployed Proxmox VE to run virtual machines for Linux and Windows Server testing.

### Ubuntu Server Deployment

Installed and configured Ubuntu Server as a Linux administration environment.

### Windows Server 2016 Deployment

Configured Windows Server 2016 for domain services, virtualization testing, and infrastructure administration.

### Cisco Switch Recovery

Recovered a Cisco Catalyst switch from bootloader mode using serial console access, PuTTY, flash initialization, BOOT variable repair, and manual IOS boot procedures.

### Hardware Refurbishment

Installed and tested SSDs, RAM, GPUs, and other components across multiple enterprise desktop systems.

## Goal

The goal of this lab is to build practical infrastructure experience comparable to entry-level systems administration, data center operations, and network support work.
